# dotfiles

## Location
- clone to home directory

## Components(neofetch)
- OS: Ubuntu
- DE: Cinnamon
- WM: Mutter

## Installation
- dotfile installation
```zsh
./install.sh
```
- vim plugin installation
```vim
:PlugInstall
```

## Setting
- CapslockをCtrlにする
    - tweaksを開いて
    - keyboard -> Additional Layout Options -> Ctrl Options
